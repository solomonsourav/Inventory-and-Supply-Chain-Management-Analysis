# Inventory & Supply Chain Management Analysis

## Overview
A Power BI dashboard analyzing inventory trends and supply chain performance.

## Files
- 📊 `Inventory and Supply Chain Management Analysis.pdf` — View the full dashboard report
- 📁 `Inventory_and_Supply_Chain_Management_Analysis.pbix` — Original Power BI file (requires Power BI Desktop)

## How to Use
### Option 1: View the PDF (No software needed)
1. Click the PDF file in the repository
2. It will open directly in your browser

### Option 2: Interactive Power BI File
1. Download the `.pbix` file
2. Open with [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Refresh data sources as needed

## Key Insights
- Inventory trend analysis across product categories
- Supply chain performance metrics
- Demand forecasting and stock level monitoring

## Tools Used
- Microsoft Power BI
